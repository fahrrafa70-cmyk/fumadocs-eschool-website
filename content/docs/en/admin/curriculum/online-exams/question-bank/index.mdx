### Add New Question

Click the **+ Question** button at the top of the table to add a new question to that Question Bank.

---

### Main Functions of Question Bank:

- As a question repository that can be used repeatedly in various exams.
- Facilitate searching and grouping questions based on subjects and categories.
- Enable centralized & efficient question input process.

---

### Brief Question Bank Navigation Flow:

1. Enter **Question Bank** menu.
2. Select **Subject** to manage.
3. Select **Question Bank** category (Essay/Multiple Choice/etc).
4. Manage questions on **Question List** page (Add/Edit/Delete).
