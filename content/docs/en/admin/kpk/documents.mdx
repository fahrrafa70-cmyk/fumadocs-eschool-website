The **KPK Document** system provides a structured file management workspace for department heads of vocational programs. Each KPK (Kepala Program Kejuruan) has access to their own document space, tailored to their specific program.

## Overview

| Item | Description |
|------|-------------|
| **Purpose** | Manage documents specific to vocational program departments |
| **Required Permission** | `dokumen-kpk` |
| **Route** | `/kpk/cards` (main hub), `/kpkfolder` (document folder) |

---

## KPK Program Structure

In Indonesian vocational schools (SMK), the KPK module typically supports multiple vocational programs such as:

- **TKJ** (Teknik Komputer dan Jaringan / Computer Networking)
- **RPL** (Rekayasa Perangkat Lunak / Software Engineering)
- **Metro** (Multimedia & Broadcasting)
- **ELIN** (Elektronika Industri / Industrial Electronics)
- And other vocational programs specific to the school

Each program head manages their own folder within the KPK system.

---

## Accessing KPK Documents

<Steps>
  <Step>
    Navigate to **KPK** from the main sidebar
  </Step>
  <Step>
    The KPK dashboard will display cards for each vocational program
  </Step>
  <Step>
    Click on your program card to open the document management area
  </Step>
  <Step>
    Browse, upload, and organize documents within your program's folder
  </Step>
</Steps>

## Setting Up KPK Programs

<Callout type="info">
  KPK program departments are configured by the School Admin from **Settings → KPK Settings**. Contact your School Admin to add or remove vocational programs.
</Callout>

## Document Categories (Suggested)

| Category | Examples |
|----------|---------|
| **Curriculum Documents** | Competency frameworks, teaching syllabi |
| **Student Performance** | Industry internship reports, competency test results |
| **Industry Partnerships** | MoU documents, industry visit records |
| **Program Administration** | Department meeting minutes, work plans |

---

## Tips

- Keep documents organized by academic year
- Use descriptive folder names that make searching easy
- Regularly back up important department documents
- Share the document link with relevant teachers in your program
