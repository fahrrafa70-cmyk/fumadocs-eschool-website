The **Online Exams** page is the control center for teachers to create, schedule, and monitor online-based exams (online) for students at every level and subject. This feature is designed to facilitate the learning evaluation process quickly, measurably, and digitally documented.

### üõ†Ô∏è Online Exam Form Filling Guide

1. **Class Level (Required)**\
   Select the class level such as X, XI, XII.
2. **Class (Required)**\
   Select a specific class, for example: XII TKJ B.
3. **Subject (Required)**\
   Select the subject, for example: MULOK PJOK - Theory.
4. **Exam Title (Required)**\
   Enter the exam name, such as: *Odd Semester Final Exam - PJOK*.
5. **Exam Key (Required)**\
   Enter the access code so only certain students can open the exam.
6. **Duration (Required)**\
   Fill in the duration in minutes, example: 60.
7. **Start Date and Time (Required)**\
   Determine the time when the exam will start.
8. **Active Exam List**\
   Displays all exams that have been created in a table format with information:
   * No
   * Class
   * Subject
   * Title
   * Total Questions
   * Start Date
   * Action (Edit / Delete / View Detail)

‚úÖ Click the **Save** button to publish the exam. Click **Close** if you want to cancel without saving.

### üõ†Ô∏è Common Cases That May Occur

| Problem                        | Solution                                                                     |
| ------------------------------ | -------------------------------------------------------------------------- |
| Subject does not appear        | Make sure the class is selected first                               |
| Exam cannot be saved           | Re-check if all fields (title, key, duration, date) have been filled |
| Questions do not appear in list | Make sure questions have been added in the question bank, or check synchronization    |

***

### üí° Additional Tips

* Use a clear and consistent exam title for easy recognition.
* Set the execution time according to the learning schedule to avoid conflict.
* Save the exam key carefully to maintain access security.
* After creating an exam, make sure students know the schedule and instructions clearly.
* Avoid editing an exam while it is already running, unless absolutely necessary.
