The Online Exam Results menu functions to display a list of scores from online exams that have been completed by students in your classes. This page is read-only (for viewing results only), so users cannot make changes to score data on this page.

---

### üîç How to Access Exam Results

To view student exam results in your class, follow these steps:
1. Go to the **Online Exams** ‚Üí **Online Exams** menu.
2. Find the exam agenda in the **Exam List** table.
3. Click the three-dot icon (**‚ãÆ**) in the **Action** column.
4. Select the **Results** menu. You will be redirected to the score recap page for that exam.

---

### Information Displayed

On this page, the system will display the following details:

| Data | Description |
| :--- | :--- |
| **Class Name** | The class taking the exam. |
| **Subject** | The subject of the exam taken. |
| **Exam Title** | The name or title of the exam that was previously created. |

---

### Exam Results List

At the bottom, there is a **Results List** table that displays:

- **Sequential Number**
- **Student Name**  
  List of names of students who have taken the exam.
- **Score**  
  Final score obtained by students from the exam (format: Score Obtained / Maximum Score).

The **Search** feature is also available to facilitate quick searching of student names, especially when the number of student data is quite large.

---

### Important Notes:

- This page is only used to view and monitor exam results.
- There are no edit or score manipulation features on this page.
- To make changes or corrections to scores, this process must be done through features available in other exam management menus (such as Correction).

---

### Functions of This Page:

- As monitoring of final online exam results that have been completed.
- Helps Teachers to manually recap scores if needed.
- As a quick reference for Teachers to validate student exam result data.
