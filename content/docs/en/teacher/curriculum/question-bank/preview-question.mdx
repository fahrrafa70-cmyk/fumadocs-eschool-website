The **Preview Question** feature allows Teachers to simulate the appearance of a question before it is actually tested on students. This step is crucial to ensure image layout, clarity of instructions, and functionality of answer options are working perfectly.

---

### How to Access

1. In the **Question List** table, find the target question row.
2. Click the three-dot icon (**â‹®**) in the **Action** column.
3. Select the **Preview** option.

---

### Features in the Preview Window

In preview mode, you can utilize several controls:
- **Student View**: See the text format and image position in real-time.
- **Answer Simulation**: You can try selecting one of the options to see how the system responds.
- **Reset Button**: Clears the simulated answer choices to try again.
- **View Answer**: A teacher-only feature to double-check the set answer key.
