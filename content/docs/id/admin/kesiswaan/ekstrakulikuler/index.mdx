## Ringkasan Alur Kerja

Untuk menjalankan modul ini secara efektif, ikuti alur kerja berikut:

1. **Definisikan Kegiatan:** Daftarkan nama ekstrakurikuler dan tunjuk Guru Pembina di menu [Kelola Ekstrakurikuler](./manage-extracurricular).
2. **Atur Jadwal:** Tentukan hari dan jam pertemuan di menu [Jadwal Kegiatan](./extracurricular-schedule).
3. **Approval Anggota:** Proses pendaftaran siswa yang masuk dari mobile app di menu [Daftar Anggota](./extracurricular-members).
4. **Catat Kehadiran:** Tanda tangani daftar hadir setiap kali pertemuan selesai di menu [Input Absensi](./extracurricular-attendance).
5. **Monitoring:** Evaluasi keaktifan siswa setiap akhir bulan di menu [Laporan & Rekap](./extracurricular-attendance-report).
