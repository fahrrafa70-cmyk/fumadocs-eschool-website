Sistem **Dokumen KPK** menyediakan ruang kerja manajemen berkas terstruktur untuk kepala program kejuruan.

## Ringkasan

| Item | Keterangan |
|------|------------|
| **Tujuan** | Mengelola dokumen spesifik departemen program kejuruan |
| **Izin yang Diperlukan** | `dokumen-kpk` |
| **Rute** | `/kpk/cards` (hub utama), `/kpkfolder` (folder dokumen) |

---

## Mengakses Dokumen KPK

<Steps>
  <Step>
    Buka **KPK** dari sidebar utama
  </Step>
  <Step>
    Dashboard KPK akan menampilkan kartu untuk setiap program kejuruan
  </Step>
  <Step>
    Klik kartu program Anda untuk membuka area manajemen dokumen
  </Step>
  <Step>
    Telusuri, unggah, dan atur dokumen dalam folder program Anda
  </Step>
</Steps>

---

<Callout type="info">
  Program-program KPK dikonfigurasi oleh Admin Sekolah dari **Pengaturan → Pengaturan KPK**. Hubungi Admin Sekolah untuk menambahkan atau menghapus program kejuruan.
</Callout>

---

## Kategori Dokumen yang Disarankan

| Kategori | Contoh |
|----------|--------|
| **Dokumen Kurikulum** | Kerangka kompetensi, silabus pengajaran |
| **Kinerja Siswa** | Laporan PKL, hasil uji kompetensi |
| **Kemitraan Industri** | Dokumen MoU, catatan kunjungan industri |
| **Administrasi Program** | Notulen rapat, rencana kerja |
