## Alur Kerja Guru Pembina

Untuk menjalankan tugas sebagai pembina secara efektif, ikuti alur kerja berikut:

1. **Kelola Jadwal:** Tentukan hari dan jam pertemuan rutin di menu [Jadwal Kegiatan](./teacher-extracurricular-schedule).
2. **Approval Anggota:** Periksa dan penuhi permintaan pendaftaran siswa di menu [Daftar Anggota](./teacher-extracurricular-members).
3. **Catat Kehadiran:** Selalu isi daftar hadir setiap kali pertemuan selesai di menu [Input Absensi](./teacher-extracurricular-attendance).
4. **Monitoring:** Evaluasi keaktifan siswa Anda di menu [Laporan & Rekap](./teacher-extracurricular-attendance-report).
